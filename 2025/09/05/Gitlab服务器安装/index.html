Gitlab服务区部署

Gitlab服务区部署

安装相关依赖

sudo apt install -y postfix #安装邮件服务器 武汉平台 sfadmin-virtual-machine
sudo apt install curl
sudo apt install openssh-server
sudo apt install ca-certificates

安装gitlab

离线安装

下载gitlab-ce_15.9.3-ce.0_arm64.deb
https://packages.gitlab.com/gitlab/gitlab-ce/
注：ubuntu focal版本

2.2 在线安装

curl https://packages.gitlab.com/install/repositories/gitlab/gitlab-ce/script.deb.sh | sudo bash
sudo apt-get install gitlab-ce

配置

sudo gitlab-ctl stop
sudo vim /etc/gitlab/gitlab.rb
external_url ‘http://192.158.11.243:4543‘
firewall-cmd –zone=public –add-port=4543/tcp –permanent

sudo gitlab-ctl reconfigure
sudo gitlab-ctl restart

系统账户管理

查看账户密码
sudo vim /etc/gitlab/initial_root_password

cd /opt/gitlab/bin

恢复

获取备份文件

根据备份文件安装指定版本gitlab

加载备份文件

sudo backfile=1673231242_2023_01_09_15.5.4_gitlab_backup.tar
sudo cp ${backfile} /var/opt/gitlab/backups/
sudo chown git:git /var/opt/gitlab/backups/${backfile}
sudo chmod g-rwx,o-rwx /var/opt/gitlab/backups/${backfile}
sudo cp gitlab.rb /etc/gitlab/
sudo cp gitlab-secrets.json /etc/gitlab/
sudo chown root:root /etc/gitlab/gitlab-secrets.json /etc/gitlab/gitlab.rb
sudo chmod g-rwx,o-rwx /etc/gitlab/gitlab-secrets.json /etc/gitlab/gitlab.rb

暂停git服务

sudo gitlab-ctl stop unicorn
sudo gitlab-ctl stop sidekiq

开始备份

gitlab-rake gitlab:backup:restore BACKUP=/var/opt/gitlab/backups/${backfile}
6、重置、重启与自检
gitlab-ctl reconfigure
gitlab-ctl restart
gitlab-rake gitlab:check SANITIZE=true
